# llamecoders
Experiments in Agentic Programming

## Setup

The setup/ directory has a `docker-compose.yaml` and a litellm config file. These are examples and you will need to adapt them for your setup.

In particular:
- the docker compose yaml brings up traefik bound to a local address, you'll need to change that.
- the litellm config file specifies gpt and infomaniak as providers. You should add your own.
- the sample opencode.json needs a valid api key
